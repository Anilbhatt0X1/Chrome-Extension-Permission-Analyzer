How to use Chrome Analyzer
  1. Paste IDs or Web Store URLs into the left text box — one per line, comma-separated, or mixed.
  2. Upload a .txt / .csv file — IDs from both sources are merged and deduplicated.
  3. Click ▶ Start Scan — results appear live as each extension is fetched directly from Google.
  4. Search by ID, name, version, description, host permission or content script domain. The Flags column highlights all matching entries with Hhost and CScontent script badges.
  5. Download ZIP on any row to save that extension's full source code.
  6. Save Report exports all results as a self-contained HTML file.
